## How to be listed here

This project welcomes contributions. Open a pull request against the repository and, if it
lands, add yourself to the list above in the same change. Keep the entry to a name and an
optional GitHub handle. See CONTRIBUTING.md for the workflow, the test commands and the
style rules.
